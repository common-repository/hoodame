A personal widget with shows users avatar, name and some metainformation.

== Description ==

Generate a personal HoodaMe widget for blog members and non members containing users name, role and some  information. 
<br/>
Supports the <a href="http://wordpress.org/extend/plugins/user-photo/">User Photo</a> plugin. 

== Installation ==

1. Extract the folder "hooda-me" from the zip file and upload it to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in the WordPress admin interface
3. Set up you're widget settings.
